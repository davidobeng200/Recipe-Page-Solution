### What I learned

I learned how to use media queries to tailor my work to different screen sizes:


```css
@media screen and (min-width: 480px) and (max-width: 1250px){
    body{
        padding: 35px;
    }

    img{
        width: 100%;
    }

    p, li{
        font-size: 20px;
    }

    h3{
        font-size: 25px;
    }

    h4{
        font-size: 20px;
    }

    tr td{
        font-size: 20px;  
    }

}

/* Mobile Screen */
@media screen and (max-width: 480px){
    body{
        padding: 35px;
```
